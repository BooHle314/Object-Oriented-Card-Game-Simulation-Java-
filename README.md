# Object-Oriented-Card-Game-Simulation-Java-
This project implements an object-oriented card game system in Java using abstraction and inheritance. A generic CardGame abstract class is created to represent a standard 52-card deck with functionality to shuffle and manage card dealing.The system is extended through multiple child classes representing different card games, each defining its own rules for the number of cards dealt and game-specific behavior. An application class demonstrates the functionality by simulating multiple card games and validating core methods such as deck initialization, shuffling, and dealing. This project demonstrates key OOP principles including abstraction, inheritance, and polymorphism.
